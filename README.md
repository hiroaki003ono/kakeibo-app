# 家計簿アプリ (Kakeibo App)

収支を記録・管理するWebアプリケーションです。 
未経験からエンジニア転職を目指す学習の成果物として、機能を段階的に追加しながら開発しています。

## 使用技術

- HTML / CSS
- JavaScript
- PHP（予定）
- MySQL（予定）

## 機能一覧

### 実装済み
- 開発中

### 実装予定
- 収支の入力・一覧表示
- カテゴリ別分類
- 月別グラフ表示（Chart.js）
- ユーザーログイン機能
- セキュリティ対策（XSS・SQLインジェクション）
- 外部データ連携（CSV取込）
