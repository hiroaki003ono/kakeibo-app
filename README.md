# 家計簿アプリ (Kakeibo App)

収支を記録・管理するWebアプリケーションです。 
未経験からエンジニア転職を目指す学習の成果物として、機能を段階的に追加しながら開発しています。

## 使用技術

- HTML / CSS
- JavaScript
- PHP（予定）
- MySQL（予定）

## 機能一覧

### 実装済み
- [x] 収支の入力フォーム
- [x] 収支の一覧表示
- [x] 収入・支出・残高の合計計算
- [x] 収支の削除機能
- [x] 入力バリデーション

### 実装予定
- 月別グラフ表示（Chart.js）
- ユーザーログイン機能
- セキュリティ対策（XSS・SQLインジェクション）
- 外部データ連携（CSV取込）
